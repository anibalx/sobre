write 
> Envia para alguém específico
```
  write <USER_A_RECEIVE> /dev/pts/<NUM>
```

wall
> Envia para todo conectados a um terminal
```
  wall <MENSAGEM>
```

**OBS**
O usuário a receber a mesagem, deve permitir o recebimento de mensagens
```
  mesg s
```
