# Base
```
argospm -h
```  

## Update
```
argospm update
```  

## Search
```
argospm search
```  

## Install
```
argospm install translate-en_pt
```  

## List
> List installed packages
```
argospm list
```  

## Remove
```
argospm remove translate-en_pt
```
