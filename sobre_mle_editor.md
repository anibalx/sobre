## Teclas
```
  <F2> = help

  (C) CTRL + r = pesquisa interativa (correlaciona os elementos encontrados pela pesquisa)
  (C) CTRL + t = replace
  (C) CTRL + f = fuzzy

  (M) ALT + j = salta entre linhas
  (M) ALT + a = seleção
  (M) ALT + = -> salta entre parênteses
  (C) CTRL + v = salta pelas correlações dos termos (não é case sensitive)

  (C) CTRL + \ = nova linha

  (C) CTRL + o = abrir arquivo

  (M) ALT + e = abrir shell

  (C) CTRL + b = abrir browser (estrutura de diretórios)

  (M) ALT + f = mova até a próxima palavra
  (M) ALT + b = mova até a palavra anterior
  (M) ALT + g = mova até a linha

  (M) ALT + ' (letra) = move para a próxima ocorrência da letra
  (M) ALT + ; (letra) = move para a ocorrência anterior da letra

  (M) ALT + 9  = linha do cursor será o topo da visualização do editor
  (C) CTRL + l = linha do cursor será usada como delimitador de visualização
  (M) ALT + x + l = linha na qual o cursor se encontra define a orientação da tela

  home | CTRL + a = move para o início da linha
  end  | CTRL + e = move para o final da linha

  (M) ALT + k = copia
  (C) CTRL + k = corta
  (C) CTRL + u = cola
  (C) CTRL + s = salva
  (M) ALT + s = salva como
  (C) CTRL + z = refazer


  (M) ALT + n = avança entre abas (buffers)
  (M) ALT + p = retorna entre abas (buffers)

  (M) ALT + c = fecha aba

  (M) ALT + v = vsplit
  (M) ALT + h = split

  (M) ALT + q = grep

  (M) ALT + . = identação
  (M) ALT + , = retira identação


  (M) ALT + \ = início do arquivo
  (M) ALT + / = fim do arquivo

  (C) ALT + c = fecha tudo
  (C) CTRL + x = fecha o programa (sair)


  (C) ALT + l = abre o comando less
```
