Aqui está uma explicação sobre o que é uma API:

## O que é uma API?

**API** significa **Application Programming Interface** (Interface de Programação de Aplicações). É um conjunto de regras e protocolos que permite que diferentes softwares se comuniquem entre si.

### Analogia simples

Pense na API como um **garçom em um restaurante**:
- **Você** (o aplicativo) faz um pedido
- O **garçom** (a API) leva seu pedido até a cozinha
- A **cozinha** (o servidor) prepara o que você pediu
- O **garçom** traz de volta a **resposta** (os dados)

Você não precisa saber como a comida foi preparada, apenas faz o pedido e recebe o resultado.

### Como funciona na prática

Quando você usa um app de previsão do tempo:
1. O app **solicita** dados à API do serviço meteorológico
2. A API recebe seu pedido, busca as informações
3. A API **retorna** os dados formatados para o app
4. O app exibe a previsão para você

### Tipos comuns de API

- **APIs Web** (REST, GraphQL) - usadas na internet
- **APIs de sistema operacional** - para acessar recursos do dispositivo
- **APIs de bibliotecas** - em linguagens de programação

### Por que APIs são importantes?

- Permitem **integração** entre sistemas diferentes
- **Aumentam a produtividade** (não é preciso reinventar a roda)
- **Isolam complexidades** - você usa a funcionalidade sem precisar entender os detalhes internos
- **Garantem segurança** - controlam o que pode ser acessado

Em resumo: APIs são os "contratos" que permitem que programas conversem entre si de forma padronizada e segura.
